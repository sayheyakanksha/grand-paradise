Last Updated: December 2025

GrandParadise operates as a U.S.-based gaming news website. This document explains how data is collected, used, and protected. The information below reflects standard practices for digital media platforms that rely on analytics and advertising.

## 1. Information We Collect

### 1.1 Automatically Collected Data

When users visit the site, certain data is gathered automatically, including:

- IP address
- Browser type and version
- Device type
- Pages viewed and time spent
- Referral URLs
- General location (city/state, not precise coordinates)

This data supports analytics, performance optimization, and security monitoring.

### 1.2 Voluntarily Provided Data

If users submit forms (contact, feedback, or newsletter), the site may collect:

- Name
- Email address
- Any message content provided

This information is only used for communication initiated by the user.

## 2. Cookies and Tracking Technologies

GrandParadise uses cookies, local storage, and similar technologies to:

- Remember user preferences
- Measure traffic and engagement
- Improve site functionality
- Support advertising partners (see Section 4)

Users can control cookie settings through their browser.

## 3. How Data Is Used

Collected data serves the following purposes:

- Operating and securing the website
- Improving content quality and user experience
- Identifying technical issues
- Responding to user inquiries
- Running analytics to understand audience behavior
- Supporting advertising systems

## 4. Advertising and Third-Party Partners

The site displays ads. Third-party ad networks may use cookies or tracking tools to:

- Deliver personalized or contextual ads
- Measure ad performance
- Prevent fraud and abuse

These partners may collect browsing data across websites to improve ad relevance. Users can opt out of interest-based advertising via their browser or device settings.

## 5. Data Sharing

GrandParadise does not sell personal information.  
Data may be shared only with:

- Analytics services
- Advertising partners
- Hosting and security providers
- Authorities if legally required

Each partner receives only what is necessary for their function.

## 6. Data Security

Standard safeguards are used to protect data from unauthorized access or misuse. No system is flawless, but reasonable measures are maintained to reduce risk.
